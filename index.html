Smart Home Automation System

Technologies: ESP32 | C++ | FreeRTOS

This academic project focused on building a connected smart-home system capable of controlling lights and a fan through gesture recognition and automatic environmental sensing. The system monitors temperature and humidity in real time and supports both automatic and manual operation via a local web interface.

My contribution:

I was responsible for assembling the circuit and developing the C++ firmware coordinating all modules through FreeRTOS tasks. I implemented DHT11 sensor functionality, ensured stable communication, accurate sensor readings, and responsive system behavior.

ChaseCart (Capstone Project – In Progress)

Technologies: Raspberry Pi 4B | ESP32 | LIDAR | RFID | IMU | C++ | Python

ChaseCart is an ongoing team-based capstone project aimed at developing an autonomous smart shopping cart capable of following paths, avoiding obstacles, and verifying its position using RFID tags. The system integrates multiple sensors and microcontrollers for navigation and object detection.

My contribution:

I am responsible for the hardware assembly and integration of the ESP32 and motor driver circuits, as well as the sensor interfacing (LiDAR, ultrasonic, and RFID). I am also contributing to the embedded software development, focusing on the communication layer between the Raspberry Pi and ESP32 for command execution and feedback control. My tasks include debugging serial communication, testing motor control routines, and refining obstacle detection behavior.

Traffic Light Controller

Technologies: VHDL | Quartus

This project involved designing a traffic light control system using finite-state machine (FSM) logic to manage two intersecting roads. The system gives priority to the main road and activates the sub-road lights dynamically when vehicles are detected.

My contribution:

I designed the FSM architecture in VHDL, defined timing transitions for each state, and simulated the design in Quartus to verify correct sequencing.
